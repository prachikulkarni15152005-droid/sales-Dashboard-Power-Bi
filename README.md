Project Overview
Built an interactive Power BI Sales Dashboard to analyze sales performance using a clean sales dataset. The dashboard provides insights into revenue trends, product categories, customer age groups, and regional sales performance.
Key Features
Interactive sales dashboard
Revenue analysis by month
Revenue by state (map visualization)
Product category analysis
Customer age group analysis
Average revenue by state
Donut, bar, line, map, and table visualizations
Tools Used
Power BI
Microsoft Excel (or CSV, depending on your dataset)
Dataset
Used a clean sales dataset for visualization and business intelligence. Since the dataset was already clean, the focus of the project was on data modeling, KPI creation, and dashboard design rather than data cleaning.
